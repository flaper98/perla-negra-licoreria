# Perla Negra - Landing Page

Landing page moderna para licores mayoristas con pedidos por WhatsApp.

## Instalar
npm install

## Ejecutar local
npm run dev

## Deploy en Vercel
Sube este proyecto a GitHub o importa el ZIP en Vercel. El comando de build es `npm run build`.

## Cambiar WhatsApp
Edita `data/products.js` y modifica `WHATSAPP_NUMBER`.
